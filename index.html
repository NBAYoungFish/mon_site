Définition du tcp ip

Le tcp/ip(transmition control protocol/internet)est une suite de protocol utiliser pour interconnectés les différents dispositifs de réseaux internet. Ces protocols sont parfois utiliser comme protocol de communication dans un réseau local(LAN).
